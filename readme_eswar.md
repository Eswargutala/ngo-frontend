# Project Understanding - Eswar

## Overview

This project is a frontend application built using Next.js. It is part of a monorepo structure where the frontend is located inside the `packages/frontend` folder.

## Technologies Used

- Next.js (React framework)
- TypeScript
- Tailwind CSS
- Radix UI components
- Zustand (state management)
- Axios (API calls)

## Project Structure

- `packages/frontend` → Main frontend application
- `src/app` → Contains pages (Next.js routing)
- `src/components` → Reusable UI components
- `src/styles` → Styling files

## Functionality

- Displays UI pages using Next.js routing
- Uses reusable components for better structure
- Handles forms using react-hook-form
- Uses modern UI libraries for better design

## My Understanding

This is a modern frontend project structured in a scalable way using a monorepo. It separates concerns using components and uses modern tools like Tailwind and Zustand for better performance and maintainability.
